# Commands

- `init` - initialize a new todo list
- `add` - adds a task
- `del` - delete a task
- `tog` - toggle a task
- `read` - reads the file and write to the current terminal

## Setup

1. Clone the repository

```sh
git clone https://github.com/democraz20/taskmd && cd dino
```

2. Restore/install packages

```sh
dotnet restore
```

3. Run with command

```sh
dotnet run [command]
```
