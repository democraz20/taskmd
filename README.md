# Task.md 
Task.md is a CLI todo list manager made with c#<br/>
preview : <br/>
![task](images\task.png "preview")

# Commands
* init - initialize a new todo list
* add - adds a task
* del - delete a task
* tog - toggle a task
* write - writes list to current terminal
